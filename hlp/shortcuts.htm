Key commands

Alt+F4 Exit
F1 Help
Ctrl+N Create new file
Ctrl+O Open file
Ctrl+S Save file
Ctrl+X Cut
Ctrl+C Copy
Ctrl+V Paste
Del Delete selection / selected note
Ctrl+F1 Decrease notes
Ctrl+F2 Increase notes
Ctrl+F3 Decrease octaves
Ctrl+F4 Increase octaves
Return Play pattern / stop play
Shift+Return

Play pattern looped
Ctrl+Return Play single row
Space Select edit / normal mode
F2 - F9 Select octave 0 - 7
Shift+Right Jump one frame forward
Shift+Left Jump one frame backward
+ Next pattern / increase instrument / increase effect parameter
- Previous pattern / decrease instrument / decrease effect parameter
F12 Kill sound
