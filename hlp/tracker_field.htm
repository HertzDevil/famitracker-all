The pattern field

The pattern field is made of all avaliable channels, where each channel by default have four columns.

Columns

  • 1. Note column
    The note and octave to play.
  • 2. Instrument column
    The instrument to use, range 00 - 3F. May be empty for no new instrument. It is possible to change instrument without retriggering the note.
  • 3. Volume column
    The volume column, range: 1 - F. Only works for the square and noise channels.
  • 4. (5, 6, 7) Effect columns
    Used for track effects. Format is Yxx, where  Y = effect number and xx = effect parameter. A list of the effects is below.
    Additional effect columns are available by clicking the arrows in channel headers.

Use the keyboard letters to enter notes, and function keys F2 - F9 to select octave.

Example: C-4 01 F --- (Plays a C 4 note using instrument 1 and volume F, no effect is used. Note C is on the 'Q'-key)

Channels

Each channel got it name from what kind of sound the produce.
It is possible to disable channels by clicking its name on top of pattern.

Square 1 & 2

Produces square-shaped waveforms. The lowest note that is possible to play is A-0 (G-0 for PAL). Both channels are identical.

Triangle

Produces triangle-shaped waveforms. It is not possible to set volume, so the volume column have no effect on this channel.
This channel is one octave lower than the square channels and cannot produce sound below A-0 (G-0 for PAL).

Noise

Produces white-noise sound.
It is only possible to play 16 different frequencies, where the lowest frequency is played at C-0 and highest at D#1 (going out of that range will wrap).

DPCM

This one plays samples that are loaded from files, instrument effects are not used at all. It is not possible to set volume for this channel either.
It is only able to play samples at 16 different pitches, defined in the instrument settings (DPCM samples-tab).

Channel effect codes

Effects are entered in the last column, in form of Xxx. (X - Effect number, xx - Effect parameter)

Example: C-4 -- - 047, plays a C-note with 047-arpeggio. (Commonly used for "C-major"-chords)

  • 0xy - Arpeggio, x and y halfnotes above the row-note
  • 1xx - Portamento on (slide to new notes), xx = speed
  • 2xx - Portamento off (xx doesn't do anything)
  • 4xy - Vibrato (alters the frequency from a sine wave), x = speed (0 - disable), y = depth
  • 7xy - Tremolo (alters the volume from a sine wave), x = speed (0 - disable), y = depth
  • Bxx - Jump to frame xx
  • Cxx - Halt (xx doesn't do anything)
  • Dxx - Skip to next frame and start at row xx
  • Exx - Set volume (xx = 0 - 15 ) Obsolete, use the volume column instead
  • Fxx - Set speed/tempo to xx (xx < 20 sets the speed, xx > 20 sets the tempo)
  • Hxy - Trigger hardware sweep up, x = period (0 - 7 ), y = shift (1 - 7). This feature is only avaliable for the square channels.
  • Ixy - Trigger hardware sweep down, x = period (0 - 7), y = shift (1 - 7). This feature is only avaliable for the square channels.
  • Pxx - Fine pitch setting. Default xx = 80, xx > 80 increase pitch, xx < 80 decrease pitch.
Notes: E will not work if an instrument volume effect loop is used. 1 will not work if an instrument frequency effect loop is used. (Pitch, Hi-pitch and Arpeggio)
