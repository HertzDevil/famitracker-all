Pattern editor

The pattern field consists of all channels and looks like this

 

Channels

The channels in the NES are named after the type of sound they generate. Channels can be muted by pressing the name.

Square 1 & 2 

Square shaped waveform. The lowest possible note is A-0 (G-0 for PAL)

Triangle

Triangle shaped waveform. Volume cannot be changed for this channel, so it's just on or off. Like the squares, it cannot go below A-0.

Noise

White noise sound. This channel has only 16 different frequencies, notes is displayed as 0 to F.

DPCM

This channel plays delta modulated PCM samples, that are loaded from DMC-files. Volume cannot be changed.
See DPCM instruments for more information.

Columns

Each channel are then split into four different columns.

1. Note column

Note and octave to play.

2. Instrument column

The instrument to use. Can be empty to use the last used instrument. It's also possible to change the instrument without retriggering the note.

3. Volume column

Sets the volume for a channel, range 0 - F. Can only be used for square and noise.

4. Channel effects

Used for channel effects. Format is Yxx, where Y = effect number and xx = effect parameter.
See this list of available effects.

Frame editor

One set of patterns for all channels are called a frame in this program, and frames are configured in the frame editor.

Each channel are represented as a pair of number, where the number is the selected pattern slot for that frame. There is a total of 128 patterns for each channel, and the patterns aren't shared across the channels. Currently 128 frames can max be used.

To select another pattern, press the "+" - or "-" button, or the those buttons on the keyboard. Change all will change all patterns at once for the selected frame.

The other toolbar

The first part is the frame editor and is already covered.

The song settings is for setting tempo and speed, number of rows and frames in the song.

Speed and tempo can also be set in the pattern by using command Fxx. The real BPM tempo is calculated by (6 / Speed) * Tempo, which means that the tempo box is correct at speed of 6. The time base for the song is usually 60 or 50 frames per second, this may lead to different durations for each row as the average tempo is calculated. If you think this is noticable, leave tempo at 150 (125 for PAL) and only use BPMs given by the speed box to make sure all rows are given the same duration.

Edit settings: step is the number of rows to steps to move in pattern editor and key repetition disables/enables repetition of keys.

Song information is: name, artist and copyright. These fields are stored in NSFs (and also available from the NSF export dialog).

Song list is used to select the song in multi song files. Song count is entered in the module properties dialog.

The last part to the right are the instrument box. The small toolbar is used to create, delete, open, save and edit settings for instrument. Textbox are for names.
