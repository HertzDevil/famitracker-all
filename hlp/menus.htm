Menus

This section expains the menus found in the main window.

File
Edit
Module
Instrument
Tracker
View
Help

File

New
Creates new module

Open
Open famitracker modules (.FTM)

Save
Saves current module

Save As
Saves current module with a new filename

Create NSF...
Exports the module to NSF or another format

Create WAV...
Renders the module to a wave file

Import MIDI...
Imports MIDI files.

Configuration
Open the configuration window

Exit
Exits the program

Edit

Undo
Undo last action in the pattern editor

Redo
Redo last action in the pattern editor

Transpose
Transpose notes. A sub menu specifies how to transpose.

Interpolate
Interpolate selected rows in the pattern editor

Reverse
Reverses a selected area in the pattern editor

Instrument Mask
Check this item to disable insertion of instrument numbers when entering notes in the pattern editor

Replace instrument
Replace the instrument in a selection with the selected instrument

Cut
Cut selection to the clipboard

Copy
Copy selection to the clipboard

Paste
Paste from the clipboard

Delete
Delete selection

Select All
Select whole channel, if whole channel is selected: select all channels

Clear Patterns
Removes all notes and data from the patterns

Clean up
Removes unused instruments or patterns from the module.

Paste and Overwrite
Pasted data will overwrite existing data instead of inserting new rows

Paste and Mix
Pasted data will mix with existing data instead of inserting new rows

Enable MIDI
Enable/disable the MIDI input device

Module

Insert Frame
Inserts a new frame

Remove Frame
Removes selected frame

Duplicate Frame
Duplicates selected frame

Move Frame Up
Moves selected frame up one step

Move Frame Down
Moves selected frame down one step

Module Properties
Open the module properties dialog

Instrument

New
Add a new instrument. The instrument type depends on the selected channel type.

Remove
Remove selected instrument

Clone
Clone selected instrument

Load
Load an instrument from a file (.fti)

Save
Save selected instrument to a file (.fti)

Edit
Open the instrument editor for the selected instrument

Tracker

Play
Play module from top of pattern

Play Pattern
Play and loop selected pattern

Play Row
Play only one row

Stop
Stop playback of module

Edit
Toggle the edit/normal mode

Toggle channel
Mute/unmute selected channel

Solo channel
Mutes all channels but the selected one

Switch to track instrument
Selects the current instrument in selected channel when playing the module

Display DPCM state
Displays the DPCM state: sample position and delta counter

Kill Sound
Reset all sound channels

NTSC
Select the NTSC mode and default framerate of 60 FPS

PAL
Select the PAL mode and default framerate of 50 FPS

Engine Speed
Select a custom player framerate or the default framerate

View

Toolbar
Displays or hides the toolbar

Status Bar
Displays or hides the status bar

Control panel
Displays or hides the control panel

Help

Help Topics
Open the help window

Effect table
Open the help window with the effect list

Performance
Opens the performance dialog

About FamiTracker
Open the about famitracker dialog
