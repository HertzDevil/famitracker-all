NSF export

NSF (Nintendo Sound Format, invented by Kevin Horton) is a type of files used for NES music.

To create NSFs, select File/Create NSF to open the NSF export dialog.

The three text-fields are the same as in the tracker main window, for setting song name, author and copyright text.

Settings

  • Force PAL sets the PAL only flag in NSF header, indicating that the tune is an exclusive PAL-system song.
  • Enable NSF bank switching creates a bankswitched song, for songs that are larger than 32 kB.
  • Remove unused NSF banks decreases the file size by setting the bankswitch numbers. Only usable if DPCM samples are used and there is enough space between samples and music data.

Change Music data address  if you want to use music in your NES app. It tells the compiler where the music data is stored in the NES's address space. Will not affect NSF's.

Press the Write NSF -button to create a NSF from your song, or Write BIN to create one file with binary music data and one file with DPCM data for use with the music driver when you want music in NES-apps. Finally, there is this option Write PRG that creates a 32 kB file ready to be burned on an EPROM/EEPROM or some other kind of memory for playback on a real NES by using a NROM-compatible cart.
