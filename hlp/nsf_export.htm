NSF export

NSF (Nintendo Sound Format, invented by Kevin Horton) is a type of file used for NES music. To create NSFs, select File/Create NSF to open the NSF export dialog. The song information box allows you to set song name, author and copyright, as present in NSFs. These are also the text field that are found in the main tracker window.

One setting is available: create PAL song. It'll just set the PAL header flag.

Then three options are available:

  • Save NSF will create a NSF file.
  • Save BIN stores just the music data and DPCM samples in separate files, for use with the music player sources.
  • Save PRG creates a single 32 kB program file, ready to be used on an NROM-compatible cart for playback on NES hardware.
