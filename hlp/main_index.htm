FamiTracker help index

 Overview
FamiTracker is a Windows sound tracker style sequencer designed for creating NES/Famicom music. The music can be exported to NSF (Nintendo Sound Format) which is possible to play on the NES/Famicom hardware. A NSF player or NES emulator can be used to play those files on a PC/Mac.
Features
  • Full realtime emulation of the channels; using blargg's band-limiting library for high quality sound
  • Three additional track-effect columns per channel is available
  • MIDI devices support
  • NSF file export, makes it possible to use music in NES applications or play on NES
  • VRC6 & MMC5 expansion chip support
  • Source is avaliable under the GNU General Public License (GPL)
