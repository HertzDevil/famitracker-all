FamiTracker Help Index

Overview

This is a sound tracker for producing music using the same limitations as the hardware found in Famicom/NES.

The CPU found in a NES (named 2A03) has a PSG that is capable of producing sound through five different channels. The first four resembles analog waveforms (two square waves, triangle wave and noise), while the last one is playing samples. That means this tracker doesn't use samples as most of them usually do, the sound is instead synthesized.

You can export music to the NSF-format (Nintendo Sound Format). A NSF-file is simply a NES program that plays music, and that makes it possible to play them on real NES hardware.

A NSF player enables these files to be played on a PC. There are a lot of players available, but NSFplug (for Winamp) and NSFplay (standalone) are recommended.

Some features:

  • Full real-time emulation of all channels, using blargg's band-limiting library for good quality
  • Automatic PCM to DPCM conversion for DPCM samples
  • 64 instruments, each instrument have 6 different effects
  • Three additional track-effect columns per channel are available
  • NSF and music data export, makes it possible to use music in NES applications
  • MIDI devices input and output / MIDI file import
  • It's possible to create multiple tracks NSFs
  • Source is avaliable under the GNU General Public License (GPL)
