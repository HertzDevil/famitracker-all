Overview

FamiTracker is a Windows sound tracker for NES/Famicom music. Music can be exported to NSF (Nintendo Sound Format.) These same files also can be exported to NES/Famicom code to be played on hardware.  A NSF player enables these files to be played on a PC/Mac. There are a lot of players available, but NSFplug (for Winamp) and NSFplay (standalone) are recommended.

Some features

  • Full realtime emulation of the channels; using blargg's band-limiting library for high quality
  • Three additional track-effect columns per channel are available
  • MIDI device support
  • NSF and music data export, makes it possible to use music in NES applications
  • Source is avaliable under the GNU General Public License (GPL)
