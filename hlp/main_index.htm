FamiTracker Help Index

Overview

This is a tracker for producing music using the hardware found in Famicom/NES. The chip used (called 2A03) contains a PSG that consists of four analog channels (oscillators) and one digital channel (for samples).
That means this tracker doesn't use samples as most trackers usually do. Instead, it's a synth with the same limitations as a real Famicom/NES.

It's possible to export music to the NSF-format (Nintendo Sound Format). An NSF-file is simply a NES binary with the only task to play music, and that makes it possible to play them on a real NES. (Using some kind of NSF player is necessary to play them on a PC.)

Some features:

  • Full real-time emulation of all channels
  • Internal tool for PCM to DPCM converting
  • 64 instruments may be used, where each instrument have 6 different effects
  • Up to 4 track-effect columns per channel is available
  • NSF and binary music data exporting (makes it possile to use music in NES applications)
  • MIDI devices input / MIDI file importing is supported
  • Source is avaliable under the GNU General Public License (GPL)

This software is still beta, please report any bugs/comments/suggestions.
