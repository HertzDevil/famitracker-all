Effect list

 

0xy Arpeggio, fast note shifting. X = second note, Y = third note
1xx Portamento up, pitch slide upward, xx = speed (00 to disable)
2xx Portamento down, pitch slide downward, xx = speed (00 to disable)
3xx Automatic portamento, automatically slides to new notes xx = speed (00 to disable)
4xy Vibrato, turn on the vibrato effect, x = speed (0 to disable), y = depth
7xy Tremolo, turn on the tremolo effect, x = speed (0 to disable), y = depth
Bxx Jump, jump to frame xx
Cxx Halt, stop the song (xx doesn't do anything)
Dxx Skip, skip to next frame and start at row xx.
Exx Volume setting, this is obsolete, use the volume colum instead
Fxx Speed/tempo, sets the speed (xx between 0 - 19) and tempo (xx between 20 - FF)
Gxx Note delay, performs a delay before this row is triggered, xx = frames to wait
Hxy Hardware Sweep Up, triggers a hardware sweep up (only available for square) x = period (0 - 7), y = shift (1 - 7)
Ixy Hardware Sweep Down, triggers a hardware sweep down (only avaliable for square), x = period (0 - 7), y = shift (1 - 7)
Pxx Fine pitch, fine pitch offset setting, xx = offset (80 is default)
Vxx Square duty / noise mode, sets the square duty and noise mode setting, xx = 0 - 3 for square and 0 - 1 for noise
Yxx DPCM sample offset, each step adds 64 bytes
Zxx DPCM delta counter, changes the delta counter (through a write to $4011), affects volume of noise and triangle. xx is between 0 - 80
