Instruments

Instruments are used to customize the sound.
As no samples are used (except for the DPCM), no actual sound is stored in instruments.
Instead it has some controls for changing the output of each channel, called instrument effects. These effects are then connected to sequences.

A sequence is a list of values that are applied to the channel at the same rate as the frame rate, shown in the main window. Default for NTSC is 60 items / second.

Instrument list

This is the big list found to the right in the main window. To add instruments click Add instrument, and double click it or click the Edit instrument-button to edit it.

Instrument editor

The instrument editor is splitted up in two parts.

  • Instrument settings
    Select sequences for each effect, the checkbox will enable it.
  • Sequence editor
    Used to set up sequences, either directly in the editor or through the MML-macro box.
    The items-scroll will set the length of the sequence. The number is the amount of items, and the time is how long one sequence takes to run.
    Left mouse button changes single items and the right button draw lines.

The following effects are avaliable

  • Volume
    Selects the amplitude.
    It is not possible to change triangle volume, but a zero will disable the channel and non-zero will enable it. 
    Sixteen levels are avaliable.
  • Arpeggio (affects all channels)
    Produces fast note-shifting, commonly used for chords.
    Ranges from -79 to +79
  • Pith & Hi-pitch (affects all channels)
    Alters the pitch, Hi-pitch is sixteen times faster.
    -127 to 126 levels avaliable
  • Duty cycle / noise mode (affects square and noise)
    Sets the duty cycle and noise mode.
    Square: 0 : 12.5%, 1 : 25%, 2 : 50%, 3 : 75%
    Noise: 0 : 32k-bit mode, 1 : 93-bit mode

A loop-point can be set by clicking the part right below the editor.

Sequences may also be entered as absolute MML-macro strings. Enter it and press enter / click Parse. It is also possible to read relative MML strings by holding shift while clicking Parse.

The DPCM channel

This channel behaves different from the others, as it uses samples loaded from files.

To assign samples, open the instrument editor and click the DPCM samples-tab. On the left side you will find a list with loaded samples and on the right side a list with the assigned samples. The format of samples must be 1-bit delta modulation, but it's possible to import and convert wave-files. See importing PCM.

None of the instrument effects will affect this channel.

Note!
Playing a DPCM sample will decrease the output level of triangle and noise. If full volume isn't regained after a sample is finished, a note stop will fix it.
