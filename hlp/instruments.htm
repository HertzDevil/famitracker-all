Instruments

Instruments are used to alter the output of the channels.

As no samples are used (except for the DPCM), no actual sound is stored in the instruments.
Instead, it has some controls for changing the output of each oscillator (channel), called "instrument effects". Each one of these effects may be connected to a sequence (that is created in the sequence editor).
The sequence items are then applied to the channel at the same rate as the framerate. (Default is 60/second for NTSC)
(A list of the instrument effects is found at the bottom of this document.)

Instrument list

Found to the right in the main window, contains all instruments used by the song.
Instruments will be added from the lowest free number, not from a selection in the list.

To edit instruments, double-click or click the Edit instrument-button.

Instrument editor

  • Instrument settings
    Contains a list used to connect sequences for the instrument effects.
  • Sequence editor
    Used to set up the sequence lists. Each sequence item consists of two values:
    • Length, has two purposes: a positive value adds a delay before the next item is executed, a negative will jump backward in the list. The delay is counted in frames. (A NES do 60/50 frames per second (NTSC/PAL))
    • Value, is the number used by effect.

      The MML field is used for MML-style sequences. All values are absolute, and '|' defines the loop point. 
      For example: "| 0 0 4 4 7 7"  (a looped 047 arpeggio). Enter the string and press Parse to convert to a sequence.

The DPCM channel

This channel is different from the others, it uses samples loaded from files. (None of the instrument effects will affect it.)

To assign samples, open the instrument editor and click the DPCM samples -tab. On the left side you'll find a list with loaded samples, and on the right side a list with the assigned samples. The format of samples must be 1-bit delta modulation, but it's possible to import and convert wave-files. See importing PCM.

Note!
Playing a DPCM sample will decrease the output level of triangle and noise. If full volume isn't regained after a sample is finished, a note stop will fix it.

Instrument settings

  1. Volume 
    Range: 0 - 15 
    Sets the channel output volume. It isn't possible to set volume of the triangle channel, but a non-zero value will activate it. 
  2. Arpeggio
    Range -128 - 127
    Produces fast note-shifting, commonly used for chords.
  3. Pitch
    Range -128 - 127
    Alters the pitch, a negative value will lower the frequency.
  4. Hi-pitch
    Range -128 - 127
    Same as pitch, but sixteen times faster.
  5. Duty cycle / Noise
    Square: 0 - 12.5%, 1 - 25%, 2 - 50%, 3 - 25% (inverted),
    Noise: 0 - 32k-bit mode, 1 - 93-bit mode
    Square - Changes the duty cycle for the square channles.
    Noise - Changes the noise random generation mode. 1 will generate 93-bit sequences (causes noise to sound looped).
    This item does not affect other channels.
