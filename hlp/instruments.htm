Instruments

Instruments are used to define how the channels should produce the sound.
Each instrument has a number of effects. These effects may be connected to a sequence, wich defines how the effect is changing over time.

Instrument list

Found to the right in the main window, contains all instruments used by the song.
Instruments will be added from the lowest free number, not from a selection in the list.

To edit instruments, click the Edit instrument- button .

Instrument editor

  • Instrument settings
    Contains a list used to connect sequences for the instrument effects. They are explained at the bottom of this file.
  • Sequence editor
    Used to set up the sequence lists. Each sequenceitem consists of two values:
    • Length, has two purposes: a positive value adds a delay before the next item is executed, a negative will jump backward in the list. The delay is counted in frames. (A NES do 60/50 frames per second (NTSC/PAL))
    • Value, is the number used by effect.

      The MML field is used for MML-style sequences. Enter the string and press Parse to convert to a sequence.

The DPCM channel

This channel is different from the others, it uses samples loaded from files. (None of the instrument effects will affect it.)

To assign samples, open the instrument editor and click the DPCM samples-tab. On the left side you'll find a list with loaded samples, and on the right side you'll find a list with assigned samples for the selected instrument. The format of samples is 1-bit delta modulation, but it is possible to import and convert usual wave-files. See importing PCM.

Important note!
Playing a DPCM sample will decrease the output level of triangle and noise. If full volume isn't regained after the sample is done, a note stop will fix it.

Instrument settings

  1. Volume 
    Range: 0 - 15 
    Sets the channel output volume. It isn't possible to set volume of the triangle channel, but a positive value will activate the channel. 
  2. Arpeggio
    Range -128 - 127
    Produces fast note-shifting, commonly used for making chords.
  3. Pitch
    Range -128 - 127
    Alters the pitch, a negative value will lower the frequency.
  4. Hi-pitch
    Range -128 - 127
    Same as pitch, but sixteen times faster.
  5. Duty cycle / Noise
    Square: 0 - 12.5%, 1 - 25%, 2 - 50%, 3 - 25% (inverted),
    Noise: 0 - 32k-bit mode, 1 - 93-bit mode
    Square - Changes the duty cycle for the square channles.
    Noise - Changes the noise random generation mode. 1 will generate 93-bit sequences (causes noise to sound looped).
    This item does not affect other channels.
